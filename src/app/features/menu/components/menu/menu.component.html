Introducción a la API Rick and Morty

Esta aplicación utiliza la API REST de Rick and Morty para mostrar información sobre personajes, episodios y ubicaciones de la popular serie. Puedes realizar las siguientes acciones:

  • **Consultar por ID**: Busca detalles de un personaje en específico por su identificador único.
  • **Filtrar personajes**: Usa filtros como nombre, estado o especie para buscar personajes.
  • **Listar todos los personajes**: Obtén una lista completa de todos los personajes.
